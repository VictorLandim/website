# Website monorepo

Taken from the official Turborepo tailwind starter: https://github.com/vercel/turbo/tree/main/examples/with-tailwind.

Monorepo is needed so that apps can share tailwind config, eslint config, tsconfig config and common react components.

### Apps

- [photos](apps/photos/README.md) | [`victor.photos`](victor.photos)
- [web](apps/web/README.md) | [`victorlandim.com`](victorlandim.com)

Note: needs [`pnpm`](https://pnpm.io/installation).
